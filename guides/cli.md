# CLI

For using the PUZZLE CLI, you need npm and Node installed on your computer.

## 1. Install

```shell
npm i puzzlelang -g
```

## 2. Launch

### 2.1 Interactive mode

```shell
puzzle
```

```shell
> print hello
```

### 2.2 Run a file

```shell
puzzle run puzzlefile.pz
```

### 2.3 Run a script

```shell
puzzle run "print hi"
```
