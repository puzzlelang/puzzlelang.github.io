PUZZLE
Puzzle is an abstract scripting language that runs in javascript environments
and can easily be extended with custom or ready-to-use modules.
Get started

Node

npm install objy

Browser

<script src="https://cdn.jsdelivr.net/npm/objy/dist/browser.js" />

{{item.name}}


* Use the abstract language that powers 'low-code' development
* Use contributed modules from remote urls or local files. No config, code only!
* Extend PUZZLE with your own language, through a simple api
Quick start

Standalone


$ npm i -g puzzlelang
$ puzzle run file.puzzle
Browser


<script src="https://cdn.jsdelivr.net/npm/puzzlelang/puzzle.browser.js" />
<script>
    print "hello world";
</script>
CLI


$ puzzle
$> print "hello world"

Embedded (js)


puzzle.parse(`print "hello world"`)
Develop pretty much anything, anywhere
Puzzle runs on top of javascript which means, it runs where javascript runs:

Node, Browsers, Servers, Iot Devices, Desktop (Electron), Mobile (PWA), Cloud Functions, ...
